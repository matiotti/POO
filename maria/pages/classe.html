Classe

Descubra o que é uma classe na Programação Orientada a Objetos.

O que é uma Classe?

Uma classe é um modelo ou molde para criar objetos. Ela define os atributos (características) e métodos (comportamentos) que os objetos terão. No PHP, usamos a palavra-chave class para definir uma classe.

Exemplo em PHP

class Carro {
  public $cor;
  public $modelo;

  public function andar() {
    echo "O carro está andando.";
  }
}

Acima, criamos uma classe chamada Carro com dois atributos ($cor e $modelo) e um método chamado andar().
