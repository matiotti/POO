Sobrecarga de Métodos em POO

A sobrecarga de métodos é um conceito onde uma classe pode ter vários métodos com o mesmo nome, mas com diferentes assinaturas (quantidade ou tipo de parâmetros). Em PHP, a sobrecarga de métodos não é suportada nativamente como em outras linguagens, mas podemos usar o método __call() para simular isso.

Exemplo de Sobrecarga de Métodos

class ExemploSobrecarga {
  public function __call($nome, $argumentos) {
    if ($nome == "saudar") {
      if (count($argumentos) == 1) {
        return "Olá, " . $argumentos[0];
      } else {
        return "Olá, desconhecido!";
      }
    }
  }
}

$obj = new ExemploSobrecarga();
echo $obj->saudar("Maria");  // Olá, Maria
echo $obj->saudar();         // Olá, desconhecido!

No exemplo acima, o método saudar() foi sobrecarregado. Dependendo de quantos parâmetros são passados, o comportamento do método muda.
