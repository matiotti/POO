Composição em POO

A composição é uma relação entre objetos onde a vida do objeto "parte" (objeto composto) depende da vida do objeto "todo" (objeto composto). Ou seja, se o objeto "todo" for destruído, todos os objetos que ele compõe também são destruídos.

Exemplo de Composição

class Motor {
  public $modelo;

  public function __construct($modelo) {
    $this->modelo = $modelo;
  }
}

class Carro {
  private $motor;

  public function __construct($modelo_motor) {
    $this->motor = new Motor($modelo_motor);
  }

  public function info() {
    return "O carro tem um motor do modelo " . $this->motor->modelo;
  }
}

$carro = new Carro("V8");
echo $carro->info();

No exemplo acima, o motor é criado e destruído juntamente com o carro, ou seja, o motor não existe sem o carro. A relação é de forte dependência entre os objetos.
