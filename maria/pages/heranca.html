Herança em POO

Herança é um mecanismo que permite a uma classe herdar atributos e métodos de outra. A classe que herda é chamada de classe filha, e a classe que é herdada é chamada de classe pai.

Exemplo de Herança

class Animal {
  public $nome;

  public function __construct($nome) {
    $this->nome = $nome;
  }

  public function falar() {
    return "O animal faz um som";
  }
}

class Cachorro extends Animal {
  public function falar() {
    return "O cachorro late!";
  }
}

$animal = new Animal("Genérico");
echo $animal->falar(); // O animal faz um som

$cachorro = new Cachorro("Rex");
echo $cachorro->falar(); // O cachorro late!

No exemplo acima, a classe Cachorro herda os atributos e métodos da classe Animal, mas pode sobrescrever o método falar() para comportar-se de maneira diferente.
