Construtor em POO

O construtor é um método especial chamado automaticamente quando um objeto é criado. Em PHP, ele é definido com o nome __construct.

Exemplo de Construtor

class Pessoa {
  public $nome;

  public function __construct($nome) {
    $this->nome = $nome;
  }

  public function apresentar() {
    return "Olá, meu nome é " . $this->nome;
  }
}

$pessoa = new Pessoa("Maria");
echo $pessoa->apresentar();

No exemplo acima, o construtor define o nome da pessoa no momento em que o objeto é criado.
