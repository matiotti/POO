Atributos em POO

Os atributos representam as características de um objeto. Em PHP, podemos definir diferentes tipos de atributos:

1. Atributo Público

É acessível de qualquer lugar do código.

class Pessoa {
  public $nome;
}

$p = new Pessoa();
$p->nome = "João";
echo $p->nome;

2. Atributo Privado

Só pode ser acessado dentro da própria classe.

class Pessoa {
  private $idade;

  public function setIdade($idade) {
    $this->idade = $idade;
  }

  public function getIdade() {
    return $this->idade;
  }
}

$p = new Pessoa();
$p->setIdade(30);
echo $p->getIdade();

3. Atributo Estático

É compartilhado entre todas as instâncias da classe.

class Contador {
  public static $total = 0;

  public function __construct() {
    self::$total++;
  }
}

new Contador();
new Contador();
echo Contador::$total; // Saída: 2
