Agregação em POO

A agregação é um tipo de relação entre objetos onde um objeto "tem um" outro objeto, mas a vida útil dos objetos envolvidos não está atrelada. Ou seja, o objeto agregado pode existir independentemente do objeto que o agrega.

Exemplo de Agregação

class Motor {
  public $modelo;

  public function __construct($modelo) {
    $this->modelo = $modelo;
  }
}

class Carro {
  public $motor;

  public function __construct($motor) {
    $this->motor = $motor;
  }

  public function info() {
    return "O carro tem um motor do modelo " . $this->motor->modelo;
  }
}

$motor = new Motor("V8");
$carro = new Carro($motor);
echo $carro->info();

No exemplo acima, o carro possui um motor, mas o motor pode existir independentemente do carro. Se o carro for destruído, o motor ainda pode ser reutilizado em outro carro.
