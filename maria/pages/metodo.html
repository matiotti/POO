Métodos em POO

Métodos são funções definidas dentro de uma classe. Eles descrevem comportamentos que os objetos daquela classe podem realizar.

Exemplo de método

class Calculadora {
  public function somar($a, $b) {
    return $a + $b;
  }
}

$calc = new Calculadora();
echo $calc->somar(5, 3); // Saída: 8
