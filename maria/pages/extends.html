Estender (Extends) em POO

O extends é usado para criar uma relação de herança entre classes. Ele permite que uma classe filha herde os métodos e atributos de uma classe pai, possibilitando reutilização de código.

Exemplo de Estender (Extends)

class Veiculo {
  public $cor;

  public function __construct($cor) {
    $this->cor = $cor;
  }

  public function info() {
    return "O veículo é de cor " . $this->cor;
  }
}

class Carro extends Veiculo {
  public $modelo;

  public function __construct($cor, $modelo) {
    parent::__construct($cor);
    $this->modelo = $modelo;
  }

  public function info() {
    return parent::info() . " e o modelo é " . $this->modelo;
  }
}

$carro = new Carro("vermelho", "Fusca");
echo $carro->info();

No exemplo acima, a classe Carro estende a classe Veiculo e herda o método info(). A classe filha pode sobrescrever esse método, como feito no exemplo.
