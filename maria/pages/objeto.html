Objeto em POO

Um objeto é uma instância de uma classe. Ele representa uma entidade com características (atributos) e comportamentos (métodos) definidos pela classe.

Exemplo de objeto

class Carro {
  public $modelo;

  public function exibirModelo() {
    return "O modelo do carro é " . $this->modelo;
  }
}

$meuCarro = new Carro();
$meuCarro->modelo = "Fusca";
echo $meuCarro->exibirModelo();

Nesse exemplo, $meuCarro é um objeto da classe Carro. Ele tem um atributo modelo e pode executar o método exibirModelo().
